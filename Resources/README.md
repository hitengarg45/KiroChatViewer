# Resources

This folder contains assets used for building the app and DMG installer.

## Files

- **icon.png** - App icon source (1024x1024 PNG)
  - Used to generate AppIcon.icns for the app bundle
  - V3 design with no padding

- **dmg_background_source.png** - DMG background source (1536x1024 PNG)
  - Purple gradient with "Drag to Install" text and arrow
  - Auto-resized to 800x600 during DMG build

- **dmg_background.png** - Resized DMG background (800x600 PNG)
  - Generated by build_dmg.sh from dmg_background_source.png
  - Used in the final DMG installer

## Usage

The `build_dmg.sh` script automatically uses these resources when building the DMG installer.
